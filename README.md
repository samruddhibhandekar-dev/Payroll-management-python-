# Payroll Management System - Python Flask

## Project Description
Python + Flask + MySQL based Employee Payroll System. Admin can add employees and generate payslips.

## Features
- Add New Employee
- View Employee List
- Generate Payslip with HRA, DA, PF calculation

## How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Create database: Run `database.sql` in MySQL
3. Update password in `database.py`
4. Run: `python app.py`
5. Open: `http://127.0.0.1:5000`

## Tech Stack
Python, Flask, MySQL, HTML
